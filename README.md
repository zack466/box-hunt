# box-hunt
A fun terminal game based on Duck Hunt.

![box-hunt gif](box-hunt.gif)

# Features
- Amazing graphics
- Engaging audio
- Addicting gameplay

# How to Play
Make sure dependencies (ncursesw, SDL2, SDL2_mixer) are installed.

Compile with `make build` and play with `./run`.
